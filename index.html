Secure Messenger (Lite)

How to use this tool?

1. Encrypt:
Type your message in the "Input" box, select a "Key" (1-25), and press "Encrypt". You can add an optional "PIN" to lock the "Auto-Decrypt" feature on the encrypted message.

2. Manual Decrypt:
Paste the ciphertext into the "Input" box, enter the correct "Key", and press "Decrypt (Manual)".

3. Auto-Decrypt (Attack):
Paste the ciphertext into the "Input" box and press "Auto-Decrypt". The tool will try all keys (1-25) on the first 10 words. If the message is locked with a PIN (Hash), it will ask for it first. View the "Attack Log" for details.

4. Share Link / QR:
Type the original message in the "Input" box, add a "Key", and a "PIN". Press "Copy Share Link" or "Generate Shareable QR". This creates a special link that forces the receiver to enter the correct PIN to view the message.

Used to lock "Auto-Decrypt" or create secure share links.
